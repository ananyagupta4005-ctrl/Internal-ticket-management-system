About This Project

What it is

An Internal Helpdesk built as a college DSA project — pure frontend, zero backend.

Tech Used

HTML5, CSS3, JavaScript (ES6+), and the browser's localStorage API.

Concepts Demonstrated

Priority Queue, FIFO Queue, Stack, Linear Search, and Comparison-based Sort.

How to Run

Just open index.html in any modern browser. That's it.

DSA Features Explained

1. Priority Queue

Each ticket has a priority weight: High = 3, Medium = 2, Low = 1. Tickets with higher weight come first.

tickets.sort((a, b) => weight[b.priority] - weight[a.priority]);

2. Stack (Undo)

Every status change is pushed onto a stack. Undo simply pops the last action.

actionStack.push(action);
const last = actionStack.pop();

3. Linear Search

O(n) scan across all tickets matching ID, title, name or description.

tickets.filter(t => t.title.toLowerCase().includes(query));

4. Comparison Sort

Sort by priority weight or by creation date using JavaScript's built-in sort.

tickets.sort((a, b) => b.createdAt - a.createdAt);

5. localStorage Persistence

All tickets and the action stack are saved to localStorage as JSON.

localStorage.setItem('helpdesk_tickets', JSON.stringify(tickets));
